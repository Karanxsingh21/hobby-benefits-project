# 📚 Hobby Benefits Project

## 📌 Project Description
This is a simple interactive web application that allows users to manage hobby benefits and submit a contact form.

## 🚀 Features
- ➕ Add new hobby benefits
- ❌ Delete benefits with animation
- 📩 Contact form submission with alert
- 🔄 Auto-clear form after submission
- 🕒 Live date and time display
- 🎨 Modern UI with animations

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## 🔄 Git Workflow
This project follows a structured branching workflow:

dev → stage → main

- **dev**: Development work
- **stage**: Testing and verification
- **main**: Final production-ready code
